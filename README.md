<span id="_Toc465287064" class="anchor"></span>Detailed Game Description

Robbie the Robot loves running through the forest and collecting coins.
He loves it so much he can’t stop running. But he has to be careful not
to crash into the many trees in his path since they can hurt him – too
many collisions and Robbie can’t collect anymore coins.

<span id="_Toc465287083" class="anchor"></span>The enemies are Trees

forest

<span id="_Toc465287084" class="anchor"></span>Scoring

<span id="_Toc465287085" class="anchor"></span>There are gold, silver
and bronze coins which each will give you different points as you
collect them.

  <span id="_Toc465287086" class="anchor"></span>Gold Coin     <span id="_Toc465287087" class="anchor"></span>100 points
  ------------------------------------------------------------ -----------------------------------------------------------
  <span id="_Toc465287088" class="anchor"></span>Silver Coin   <span id="_Toc465287089" class="anchor"></span>50 points
  <span id="_Toc465287090" class="anchor"></span>Bronze Coin   <span id="_Toc465287091" class="anchor"></span>10 points

<span id="_Toc465287092" class="anchor"></span>Trees will cost you 20
health. If you hit 5 trees the game over screen will come up and you
will have to restart the game.
